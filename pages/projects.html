Raspberry Pi Mini Server

❮ ❯

Description

I built a low-cost Raspberry Pi-based mini server to provide real-time monitoring of system performance, including details about resource usage and health. It can be converted into a VPN, ad blocker, private cloud, or private NAS, making it a great solution for home networking and small business needs.

Working

Monitors key metrics such as CPU load, memory usage, and disk space in real-time. The server features a built-in Uninterruptible Power Supply (UPS) to ensure continuous operation, complemented by an OLED display that provides immediate feedback on system performance.

Components Used

  • Raspberry Pi 4 Model B
  • MicroSD Card (32GB or larger)
  • Power Supply (5V, 3A)
  • USB HDD or SSD for storage
  • OLED Display (optional)
  • Cooling Fan (optional)
  • Network cable (Ethernet)

Key Features

  • VPN functionality
  • Ad blocking capabilities
  • Private cloud storage
  • Real-time performance monitoring

WiFi Deauther

❮ ❯

Description

The WiFi Deauther project enables users to test the security of WiFi networks by sending deauthentication packets to disconnect devices. This tool can be useful for penetration testing and understanding the vulnerabilities of wireless networks.

Working

The WiFi Deauther sends deauthentication packets to a targeted WiFi network, disconnecting devices attempting to connect. This allows users to simulate attacks and assess network security vulnerabilities, helping to improve defenses against potential threats. It enables analysis of device responses and testing of various attack vectors on WiFi protocols.

Components Used

  • NodeMCU ESP8266
  • Battery Pack
  • Breadboard and Jumper Wires
  • Computer
  • Power Supply
  • WiFi Adapter (if necessary)

Key Features

  • Penetration testing tool
  • Wireless network assessment
  • Targeted deauthentication
